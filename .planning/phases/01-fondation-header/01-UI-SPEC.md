- Aucune taille de titre H1-H6 dans cette phase — réservées aux Phases 2 et 3

## Color

| Role | Token CSS | Valeur hex | Usage dans cette phase |
|------|-----------|-----------|----------------------|
| Dominant (60%) | `--color-background` | #FFFFFF | Fond page, fond header scrollé (via surface) |
| Secondary (30%) | `--surface` | #FCF9F5 | Header scrollé : `rgba(252, 249, 245, 0.92)` — glassmorphism |
| Accent (10%) | `--color-primary` | #E49400 | Skip link fond uniquement |
| Destructive | `--color-error` | #BA1A1A | Non utilisé dans cette phase |

**Accent réservé pour :** skip link background (`.skipLink { background: var(--color-primary) }`) — uniquement cet élément dans cette phase. Aucun autre usage de l'accent dans le header.

**États de couleur du header :**

| État | Background | Texte | Box-shadow |
|------|-----------|-------|------------|
| Transparent (sur hero) | `transparent` | `var(--color-text)` #1D1D1B | aucun |
| Scrolled (après 80px) | `rgba(252, 249, 245, 0.92)` + `blur(20px)` | `var(--color-text)` #1D1D1B | `--shadow-header` : `0 2px 12px rgba(0,0,0,0.08)` |

Note : L'état "texte blanc" documenté dans RESEARCH.md (pour fond hero sombre) dépend du logo PNG blanc (Logo-04). La couleur du texte du header reste `--color-text` dans les deux états — c'est le **logo PNG** qui change (blanc → noir), pas la propriété CSS `color`. La seule propriété `color` qui transite est héritée par `.shopifyLink` et `.skipLink`.

## Composants visuels

### Logo header

**État transparent (hero sombre) :**
- Fichier source : `fichier-mobelunique/Logo Design/Logo-04.png` → copier vers `public/brand/logo-white.png`
- Rendu via : `<Image src="/brand/logo-white.png" alt="Möbel Unique" />`
- Taille source : 4168×4167px PNG RGBA (fond transparent, motif blanc)
- Hauteur affichée : 36px (hauteur cible dans header 64px — clear space 14px haut+bas)
- `width` calculée : proportionnel — le logo est ~3:1 (monogramme + texte), soit ~108px de large pour 36px de haut
- `next/image` : `height={36} width={108}` — valeurs exactes à confirmer visuellement

**État scrolled (fond blanc) :**
- Fichier source : `fichier-mobelunique/Logo Design/Logo-01.png` → copier vers `public/brand/logo-black.png`
- Rendu via : `<Image src="/brand/logo-black.png" alt="Möbel Unique" />`
- Taille source : 4168×4167px PNG RGBA (fond transparent, motif noir)
- Hauteur affichée : 36px (identique)
- Swap conditionnel via `scrolled` state : `src={scrolled ? '/brand/logo-black.png' : '/brand/logo-white.png'}`

**Transition du swap :** Instantané (pas d'animation de fondu sur le logo lui-même) — la transition CSS du header (`300ms ease`) couvre déjà l'ensemble de la bascule visuelle. Ajouter une `opacity transition` sur l'image serait redondant et complexe le SSR.

**Attribut priority :** Ajouter `priority` sur `<Image>` — le logo est LCP candidate pour le header fixe visible immédiatement.

### Skip link accessibilité (HEAD-04)

**Comportement attendu :**
- Invisible au repos : `position: absolute; top: -100%` — hors du viewport
- Visible au focus clavier uniquement : `top: var(--spacing-md)` (16px depuis le haut)
- Background : `var(--color-primary)` #E49400, texte : `var(--color-background)` #FFFFFF
- Border-radius : `var(--radius-md)` 8px
- Font : 14px weight 600
- Outline au focus : `2px solid var(--color-text)` offset 2px (déjà dans Header.module.css)
- Cible : `href="#main-content"` — `<main id="main-content">` dans page.tsx

### Lien "Retour à la boutique"

**Contenu :** Texte exact = `Retour à la boutique`
**URL :** `https://www.mobelunique.fr/` (confirmé CONTEXT.md D-08)
**Comportement lien :** Même onglet — `<a>` sans `target="_blank"` (CONTEXT.md D-09)
**Style :** `font-size-sm` (14px) weight 500, `color: inherit`, opacity 0.85 au repos → 1 au hover
**`rel`:** `rel="noopener"` uniquement (même onglet, pas besoin de `noreferrer`)
**Aria :** `aria-label="Retourner à la boutique Möbel Unique"`

## Interactions et états

### Header — machine à états

```
État initial (SSR)
  → scrollY = 0 → TRANSPARENT
  → scrollY > 80px → SCROLLED

TRANSPARENT
  - background: transparent
  - logo: Logo-04 blanc (/brand/logo-white.png)
  - color: var(--color-text)
  - shadow: none

SCROLLED
  - background: rgba(252,249,245,0.92) + blur(20px)
  - logo: Logo-01 noir (/brand/logo-black.png)
  - color: var(--color-text)
  - shadow: var(--shadow-header)

Transition TRANSPARENT → SCROLLED : 300ms ease (var(--transition-fast))
Seuil : scrollY > 80px
Listener : scroll passif { passive: true }
```

### Responsive header

| Breakpoint | Padding horizontal |
|------------|-------------------|
| < 1024px (mobile + tablet) | `var(--container-padding-mobile)` = 24px |
| ≥ 1024px (desktop) | `var(--container-padding-desktop)` = 48px |
| ≥ 1280px (large) | `var(--container-padding-large)` = 64px |

### globals.css — ajouts de fondation

Deux propriétés à ajouter dans le bloc `html {}` existant (déjà présentes selon l'état du code lu) :
- `scroll-behavior: smooth` — ligne 97 ✓ déjà présente
- `scroll-padding-top: var(--header-height)` — ligne 98 ✓ déjà présente
