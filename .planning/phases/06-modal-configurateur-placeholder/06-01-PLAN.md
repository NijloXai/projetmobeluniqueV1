    Export nomme `ConfiguratorModal` :

    1. **Refs et state derive :**
       - `const dialogRef = useRef<HTMLDialogElement>(null)`
       - `const open = model !== null`

    2. **useEffect scroll lock iOS-safe (per RESEARCH.md Pattern 4) :**
       Quand `open` passe a true :
       - `const scrollY = window.scrollY`
       - `document.body.style.position = 'fixed'`
       - `document.body.style.top = '-${scrollY}px'`
       - `document.body.style.width = '100%'`
       Quand `open` passe a false :
       - `const scrollY = document.body.style.top`
       - Reset position, top, width a ''
       - `window.scrollTo(0, parseInt(scrollY || '0') * -1)`

    3. **useEffect dialog control (per ConfirmDialog pattern) :**
       - Si `open && !dialog.open` : `dialog.showModal()`
       - Si `!open && dialog.open` : `dialog.close()`
       Dependance : `[open]`

    4. **handleDialogClick (per RESEARCH.md Pattern 3 — backdrop click) :**
       `const handleDialogClick = useCallback((e: React.MouseEvent<HTMLDialogElement>) => { if (e.target === e.currentTarget) { onClose() } }, [onClose])`

    5. **Early return :** `if (!model) return null` — IMPORTANT : apres les hooks (React rules of hooks)

    6. **Derive :** `const imageUrl = getPrimaryImage(model.model_images)`

    7. **JSX :**
       ```

         className={styles.dialog}
         aria-modal="true"
         role="dialog"

         <div className={styles.content}>
       ```
