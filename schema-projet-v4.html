🖥️ Front-End — 3 modules

Tout sur une seule page — deux parcours d'entrée

F1 — Hero + Catalogue 7 fonctions

Hero immersif présentant la visualisation IA + catalogue de tous les canapés. Point d'entrée direct. Si accès via Shopify (?produit=slug), cette section est bypassée.

  • Hero immersif (visuel plein écran, baseline, CTA)
  • Lire les produits actifs depuis Supabase
  • Afficher la grille de canapés (cartes : image, nom, prix)
  • Aperçu swatches tissu sur chaque carte
  • Clic sur canapé → scroll vers configurateur
  • Détection paramètre URL (?produit=slug) → skip vers F2
  • Adaptation responsive (mobile first)
F2 — Configurateur 10 fonctions

Sélection tissu + rendu IA + récap prix. Deux modes : standard (inclus) et premium (+80€). Point d'entrée depuis Shopify.

  • Afficher le canapé sélectionné (photos multi-angles)
  • Lire les tissus avec rendu validé+publié
  • Afficher les swatches tissu (badge "Inclus" / "+80€")
  • Charger le rendu IA (canapé × tissu × angle actif)
  • Navigation entre angles du canapé
  • Calculer le prix (base + premium si applicable)
  • Récap : modèle + tissu + prix total
  • Bouton "Commander" → redirection fiche produit Shopify
  • Bouton "Visualiser chez moi" → scroll vers F3
  • Bouton "Changer de canapé" → retour catalogue
F3 — Simulation IA "chez moi" 7 fonctions

Upload photo du salon → Nano Banana 2 place le canapé configuré dans l'espace du client. Résultat éphémère (pas stocké).

  • Récupérer canapé + tissu du configurateur (Zustand)
  • Instructions (photo + feuille A4 pour référence taille)
  • Upload image du salon
  • Appel API Nano Banana 2 (temps réel)
  • Overlay logo Möbel Unique (watermark)
  • Télécharger l'image générée
  • Bouton "Commander" → Shopify / "Réessayer"
F5 — Layout global RETIRÉ V4

Plus de navigation multi-pages. Remplacé par un header minimal (logo + lien retour Shopify).

F6 — Pages statiques RETIRÉ V4

Plus de pages À propos, Contact, Mentions légales. Tout reste sur Shopify.

⚙️ Back-End — 3 modules

Admin + gestion + génération IA

B1 — Auth admin 4 fonctions

Connexion admin sécurisée, protection des routes back-office.

  • Formulaire de connexion (email / mot de passe)
  • Authentification Supabase Auth
  • Protection des routes admin (middleware)
  • Gestion session (connexion, déconnexion)
B2 — Produits + Génération IA 16 fonctions ⚡

CRUD canapés + photos multi-angles + génération IA. Deux modes d'ajout : classique (sans IA) ou avec génération. Module le plus lourd.

  • Afficher liste des canapés (tableau)
  • Formulaire ajout canapé (nom, prix, dimensions, lien Shopify)
  • Formulaire modification canapé
  • Upload photos multi-angles (face, profil, 3/4, etc.)
  • Toggle actif/inactif canapé
  • Suppression avec confirmation
  • Validation des champs
  • Mode ajout classique : upload photos finales sans génération IA
  • Choisir un tissu pour la génération IA
  • Lancer Nano Banana 2 (photo canapé + ref tissu → rendu)
  • Bouton "Générer tout" → lance tous les angles d'un coup
  • Bouton "Générer" par angle → relance un seul rendu
  • Preview → Valider ou Régénérer (écrase le précédent)
  • Toggle publié/masqué par rendu
  • Galerie des rendus validés par canapé
  • Export ZIP des rendus validés (télécharger pour Shopify)
B3 — Gestion tissus 8 fonctions ⚡

CRUD tissus + catégorie standard/premium. Photo de référence pour Nano Banana.

  • Afficher liste des tissus (tableau)
  • Formulaire ajout tissu
  • Formulaire modification tissu
  • Upload image swatch (texture)
  • Upload photo de référence (pour Nano Banana)
  • Catégorie : Standard (inclus) / Premium (+80€)
  • Toggle actif/inactif (global)
  • Suppression avec confirmation

⚡ Règles métier

V4 — simplifié

Visibilité côté client

Un canapé est visible si models.is_active = true
Un tissu est visible si fabrics.is_active = true ET au moins 1 rendu is_validated = true ET is_published = true
Tissu désactivé (B3) → disparaît de tous les canapés
Rendu masqué (B2) → tissu disparaît de ce canapé uniquement

Pricing V4

Tissu Standard (is_premium = false) → Prix = models.price
Tissu Premium (is_premium = true) → Prix = models.price + 80€
Le supplément est fixe à 80€, identique pour tous les modèles

Génération IA (Nano Banana 2)

Back-office (B2) : pré-généré. "Générer tout" (tous les angles) ou "Générer" (un angle). Validé → publié → exportable en ZIP.
Simulation "chez moi" (F3) : temps réel. Résultat éphémère (pas stocké en BDD).
La régénération écrase le rendu précédent.
La photo de référence est stockée sur le tissu (B3).

Lien intelligent Shopify ↔ App

Shopify → App : ?produit=canape-oslo → le configurateur charge le bon canapé via models.slug
App → Shopify : bouton "Commander" → redirige vers models.shopify_url (fiche produit)
Si le slug n'existe pas ou le produit est inactif → fallback vers le catalogue
