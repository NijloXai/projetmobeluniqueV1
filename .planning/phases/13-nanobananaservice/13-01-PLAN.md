async addWatermark(imageBuffer: Buffer, text = 'MOBEL UNIQUE -- Apercu'): Promise<Buffer> {
  const metadata = await sharp(imageBuffer).metadata()
  const width = metadata.width || 800
  const height = metadata.height || 600
  const watermarkSvg = `<svg width="${width}" height="${height}" xmlns="http://www.w3.org/2000/svg">
    <text x="50%" y="50%" text-anchor="middle" dominant-baseline="middle"
      font-family="Arial, sans-serif" font-size="36" font-weight="bold"
      fill="rgba(255,255,255,0.35)"
      transform="rotate(-30, ${width / 2}, ${height / 2})">${escapeXml(text)}</text>
  </svg>`
  return sharp(imageBuffer)
    .composite([{ input: Buffer.from(watermarkSvg), gravity: 'centre' }])
    .toBuffer()
}

**Classe `ImageSafetyError` (exportee) :**
- extends `Error`
- message = `"Cette image n'a pas pu etre traitee (contenu non autorise). Essayez une autre photo."`

**Classe `NanoBananaService implements IAService` :**

1. **Constructeur (per D-16, singleton)** :
   - Lire `process.env.NANO_BANANA_API_KEY`
   - Si absent, throw `Error('NANO_BANANA_API_KEY manquante -- service IA non configure.')`
   - Instancier `this.ai = new GoogleGenAI({ apiKey })`
   - NE PAS logger la cle API (per D-17)
   - Logger `[IA] NanoBananaService initialise (modele: ${MODEL})`

2. **`async generate(request: GenerateRequest): Promise<GenerateResult>`** (per IA-01, IA-03, IA-05, IA-07) :
   - Extraire `{ modelName, fabricName, viewType, sourceImageUrl }` de request
   - Determiner le prompt : si `viewType === 'simulation'` utiliser `buildSimulatePrompt(modelName, fabricName)`, sinon `buildBackOfficePrompt(modelName, fabricName, viewType)`
   - Appeler `this.resolveImagePart(sourceImageUrl)` pour obtenir le part image
   - Boucle retry `for (let attempt = 0; attempt < MAX_RETRIES; attempt++)` :
     - `const startTime = Date.now()`
     - Appeler `this.ai.models.generateContent()` avec :
       - `model: MODEL`
       - `contents: [{ text: prompt }, imagePart]`
       - `config: { responseModalities: ['TEXT', 'IMAGE'], abortSignal: AbortSignal.timeout(TIMEOUT_MS) }`
     - Extraire `response.candidates?.[0]`
     - Si pas de candidate : throw `Error('Aucune reponse du modele IA.')`
     - **Verifier finishReason AVANT parts** (per IA-03) :
       - Si `candidate.finishReason !== 'STOP'` :
         - Si `candidate.finishReason === 'IMAGE_SAFETY'` : throw `new ImageSafetyError()`
         - Sinon : throw `Error(\`Generation refusee (${candidate.finishReason}).\`)`
     - Chercher le part image : `candidate.content?.parts?.find(p => p.inlineData?.data)`
     - Si pas de part image : throw `Error('Reponse Gemini sans donnees image.')`
     - Decoder : `Buffer.from(imgPart.inlineData.data, 'base64')`
     - **Validation buffer (per D-03)** : si `rawBuffer.length === 0` throw `Error('Buffer image vide retourne par le modele.')`
     - **Conversion PNG vers JPEG (per IA-05)** : `sharp(rawBuffer).jpeg({ quality: 85 }).toBuffer()`
     - `const duration = Date.now() - startTime`
     - Logger (per D-13, D-14) : `console.log(\`[IA] generate OK -- model=${MODEL} attempt=${attempt + 1} duration=${duration}ms size=${imageBuffer.length}b viewType=${viewType}\`)`
     - Retourner `{ imageBuffer, mimeType: 'image/jpeg', extension: 'jpg' }`
   - **Catch (per D-02, D-05, D-06, D-15)** :
     - Si `ImageSafetyError` : re-throw immediatement (non-retryable)
     - Si erreur contient `'API_KEY_INVALID'` ou `'401'` ou `'UNAUTHENTICATED'` : throw `Error('Configuration IA invalide. Verifiez la cle API.')` (per D-15)
     - Verifier `isRetryableError(err)` : true si message contient `'429'`, `'RESOURCE_EXHAUSTED'`, `'500'`, `'502'`, `'503'`, ou `err.name === 'AbortError'`
     - Si non retryable ou derniere tentative : throw l'erreur
     - Si retryable et pas derniere tentative : calculer delay = `BASE_DELAY_MS * Math.pow(2, attempt) * (0.8 + Math.random() * 0.4)` (per D-06 : 1s/2s/4s + jitter)
     - Logger (per D-14) : `console.warn(\`[IA] Retry ${attempt + 1}/${MAX_RETRIES} dans ${Math.round(delay)}ms -- ${lastError.message}\`)`
     - `await sleep(delay)`
   - Apres la boucle : `throw lastError!`

3. **`private async resolveImagePart(sourceImageUrl: string)`** (per IA-07) :
   - Si `sourceImageUrl.startsWith('data:')` (chemin simulate) :
     - `const [meta, data] = sourceImageUrl.split(',')`
     - `const mimeType = meta.split(':')[1].split(';')[0]`
     - Retourner `{ inlineData: { mimeType, data } }`
   - Sinon (chemin admin, URL Supabase) :
     - `const res = await fetch(sourceImageUrl)`
     - Si `!res.ok` : throw `Error(\`Impossible de recuperer l'image source (${res.status}).\`)`
     - `const buffer = await res.arrayBuffer()`
     - `const data = Buffer.from(buffer).toString('base64')`
     - `const mimeType = res.headers.get('content-type') || 'image/jpeg'`
     - Retourner `{ inlineData: { mimeType, data } }`
