```typescript
'use client'
import { useState, useEffect } from 'react'

export function Header() {
  const [scrolled, setScrolled] = useState(false)

  useEffect(() => {
    const handleScroll = () => {
      const isScrolled = window.scrollY > 80
      setScrolled(prev => prev !== isScrolled ? isScrolled : prev)
    }
    window.addEventListener('scroll', handleScroll, { passive: true })
    return () => window.removeEventListener('scroll', handleScroll)
  }, [])
  // ...
}
```

**Etape 2 — Creer `src/components/public/Hero/Hero.tsx` :**

```typescript
'use client'

import { useState, useEffect } from 'react'

```

**State scroll (pattern Header.tsx) :** `const [scrolled, setScrolled] = useState(false)` — jamais `window` dans useState (SSR). useEffect avec scroll listener passif, seuil `scrollY > 0` (pas 80 comme le Header — ici on veut que l'indicateur disparaisse des le premier pixel de scroll). Cleanup dans le return.

**Reduced motion :** `const prefersReducedMotion = useReducedMotion()`

**JSX du composant :**

1. `<section className={styles.hero}>` — conteneur principal. Pas de id necessaire.

2. `<motion.div className={styles.heroContent}>` — wrapper anime avec :
   - `initial={{ opacity: 0, y: prefersReducedMotion ? 0 : 20 }}`
   - `animate={{ opacity: 1, y: 0 }}`
   - `transition={{ duration: prefersReducedMotion ? 0 : 0.4, ease: 'easeInOut' }}`
   - Ce motion.div contient badge + H1 + sous-titre + CTA (fade simultane global per D-11)

3. **Badge pill :** `<span className={styles.badge}>Visualisation par IA</span>`

4. **H1 :** `<h1 className={styles.title}>Visualisez votre canape chez vous</h1>` — texte exact per D-04

5. **Sous-titre :** `<p className={styles.subtitle}>Choisissez votre tissu, configurez votre modele et visualisez le resultat directement dans votre salon&nbsp;&mdash; avant meme de commander.</p>` — texte exact per D-05. Utiliser `\u00A0\u2014` pour le tiret em avec espace insecable en JSX, ou bien le HTML entity.

6. **CTA :** `<a href="#catalogue" className={styles.cta}>Decouvrir nos canapes</a>` — per D-07. Lien mort temporaire (pas de section #catalogue encore).

7. **Indicateur de scroll :** En dehors du motion.div (pas anime avec le fade-in), positionne en absolute dans le hero :
   ```tsx
   <div className={`${styles.scrollIndicator}${scrolled ? ` ${styles.scrollIndicatorHidden}` : ''}`}>
     <svg className={styles.chevron} width="24" height="24" viewBox="0 0 24 24" fill="none" xmlns="http://www.w3.org/2000/svg" aria-hidden="true">
       <path d="M6 9l6 6 6-6" stroke="currentColor" strokeWidth="2" strokeLinecap="round" strokeLinejoin="round" />
     </svg>
     <span className={styles.scrollText}>Defiler</span>
   </div>
   ```
   Le SVG est un chevron vers le bas simple. `aria-hidden="true"` car decoratif (per D-16).

**Points critiques :**
- Pas de `onClick` sur l'indicateur de scroll (per D-16 — decoratif uniquement)
- Le texte utilise des caracteres accentues francais : `canape` -> `canapé`, `Decouvrir` -> `Découvrir`, `Defiler` -> `Défiler`, `modele` -> `modèle`
- z-index: heroContent a `position: relative; z-index: 2` pour etre au-dessus de l'overlay `::before` (z-index: 1)
